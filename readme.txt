A block to countdown to a date or time.

== Description ==


Gutenberg count down to a date or duration.

## FEATURES ##
 - Choosing a date/duration to count down
 - Easily assign the number/date to count down to the  certain time
 - Allow user to customize the design with difference font and size
 - No coding skill required
 - Fully Customizable
 - Fully Responsive
 - Multiple count down timer on same page


== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/countdown-settimer-block` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress
